# 玉米的凑服 - 整合包


## 整合包基础信息
- 游戏版本：JAVA Minecraft 1.21.1
- 加载器：NeoForge_21.1.230


## 玩法特性
### 机械动力拓展体系
基于原版机械动力本体，搭载二十余款专属拓展模组，全方位升级机械动力玩法，覆盖能源、工业、物流、装饰、餐饮、航空、冶金全体系，告别原版单调机械玩法，打造沉浸式科技工业生存体验。
- 能源系统：柴油发电、电气时代、交错电网，实现机械能转电能，搭建全自动电力工业体系
- 物流运输：超级管道、无人机物流、铁路导航，完成物品、流体全自动远距离传输
- 工业开采：自动矿石开掘、冶金锻造、附魔工业，简化资源获取，拓展装备强化与材料加工玩法
- 趣味功能拓展：机械龙模组、内饰装饰、自动扶梯、跑步机、油炸工艺、中央厨房、主题咖啡馆，兼具功能性与观赏性
- 功能补强：创意传动、物品附加、炫彩金库，补齐原版机械动力功能短板，拓展建造与存储玩法
### 休闲农耕生存玩法
新增农夫乐事模组，拓展原版农耕、烹饪、美食体系，丰富田园生存内容；同时优化鞘翅获取方式，末影龙直接掉落鞘翅，降低探索门槛，提升飞行探索体验，兼顾休闲养老与冒险乐趣。
### 其他优化
- 查询辅助：JEI物品管理器、玉（Jade），一键查看配方、方块/实体属性
- 视觉优化：动态光源、3D皮肤层、皮肤修复、附魔描述可视化
- 探索辅助：Xaero小地图，支持路标标记、地形查看、死亡点位记录
- 联机优化：账号快速切换、聊天头像显示、禁用聊天举报，安全稳定联机
- 储物优化：精妙背包+精妙核心，大容量智能储物，解决背包拥挤问题
- 内存优化：铁氧体磁芯、现代化修复，降低内存占用，杜绝闪退、卡顿
- 渲染优化：实体渲染机制优化、动态FPS，闲置降帧、动态适配画质
- 区块优化：区块预生成、C²M区块引擎，解决远距离加载卡顿、空洞卡顿
